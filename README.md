## Run Locally

Clone the project

```bash
  git clone https://github.com/adnan-54/SLStudio.git
```

### Client

Navigate to the Client folder

```bash
  cd client
```

Run the project

```bash
  dotnet run --project studio\slstudio
```

### Web

Navigate to the Web folder

```bash
  cd web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
